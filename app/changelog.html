Changelog

v1.7 2026-05
TYPE/NAME Directory Structure
  • Video root now uses a two-level layout: TYPE/NAME/… (e.g. Drama/My Show/EP01.mp4)
  • Sidebar groups by type at the top level, then by show within each type
  • Admin permission tree updated to reflect the new three-level hierarchy (type → show → section/file)
  • Files without an EP## prefix are assigned index 01 instead of 9999
v1.6 2025-05
Changelog Page
  • Added this changelog page with feature overview, library summary, and history
v1.5 2025
Per-user Permissions
  • Admin can restrict each user to specific shows or episodes
  • VTT subtitles inherit access from their paired MP4
  • Admin library view for inspecting the video structure
v1.4 2025
Profiles & Bilingual UI
  • Avatar upload (JPG / PNG / GIF / WEBP, up to 2 MB)
  • Language toggle: English and Traditional Chinese
  • Password change from profile page
  • Meme intro with skip options (today / this week / always)
v1.3 2025
Admin Panel
  • Add and delete user accounts
  • Online / offline status and last login info per user
  • IP ban management with 15-minute lockout
v1.2 2025
Watch Progress & Subtitles
  • Watch position synced to server every 5 seconds
  • Auto-resume from saved position on revisit
  • Progress bar shown per episode in the sidebar
  • VTT subtitle track loaded automatically when file exists
v1.1 2025
Playback Controls & Mobile
  • Speed control: 1×, 1.25×, 1.5×, 2×
  • Skip buttons: ±5s, ±10s, ±1m, ±10m
  • Prev / Next episode navigation
  • Mobile layout with slide-up episode drawer
v1.0 2025
Initial Release
  • Video streaming from a local directory via Flask
  • Session-based login with brute-force protection
  • Episode list sidebar with search
  • Multi-show grouping with collapsible sections
